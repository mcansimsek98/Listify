## General info

This application is built using Firebase Authentication, Firestore Database, and Firebase Storage to enable users to create a task list. The created tasks can be categorized, have dates added to them, and can be filtered based on whether they are completed or past their due dates. Swift UI has been chosen as the preferred framework for this application.	
## Technologies and 3rd party libraries
Project is created with:
* SwiftUI
* MVVM
* Firebase
